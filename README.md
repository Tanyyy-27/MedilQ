# Overview

MedilQ is a desktop-based Medical Store Inventory Management System developed to simplify pharmacy operations through intelligent inventory tracking and supplier management.

The system enables pharmacies to efficiently manage medicine records, monitor stock availability, organize supplier information, and generate purchase reports while reducing manual effort and improving operational accuracy.
